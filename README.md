# Contest Tracker

A modern web application for tracking coding contests.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- Bun (or npm/yarn)

### Installation

```bash
bun install
```

### Development

```bash
bun run dev
```

### Build

```bash
bun run build
```

### Preview

```bash
bun run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
